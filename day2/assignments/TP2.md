### Exercise 1:

- Suppose C program containing the following statements:

    - int   i =  8;
    - int   j =  5;
    - float x =  0.005f;
    - float y = -0.01f;
    - char  c = 'c';
    - char  d = 'd';

- Determine the value of these expressions:

    - (3*i - 2*j)%(2*d - c)
    -  2*((i/5) + (4*(j-3))%(i + j - 2))
    -  i <= j
    -  j != 6
    -  c == 99
    -  5*(i + j) > 'c'
    - (i > 0) && (j < 5)
    - (i > 0) || (j < 5)
    - (x > y) && (i > 0) || (j < 5)
    - (x > y) && (i > 0) && (j < 5)
